# 🐾 PetMatch

Um "Tinder para pets" — protótipo de adoção responsável onde o usuário desliza cards de animais e dá *match* com o futuro melhor amigo.

Primeiro modelo (MVP visual) em **HTML + CSS + JavaScript puro**, sem dependências, pronto pra rodar e publicar no GitHub Pages.

## ✨ O que já funciona

- Pilha de cards com gesto de arrastar (swipe) por mouse e toque
- Carimbos de **Adotar** / **Passar** durante o arraste
- Botões de ação: curtir, passar, voltar (rewind) e detalhes
- Tela de **match** com animação
- Estado vazio e botão de recomeçar
- Layout responsivo (foco mobile) e suporte a `prefers-reduced-motion`

## 🚀 Como rodar

Basta abrir o arquivo `index.html` no navegador. Não precisa instalar nada.

### Publicar no GitHub Pages
1. Suba os arquivos no repositório.
2. **Settings → Pages → Branch: `main` / root**.
3. Acesse o link gerado.
